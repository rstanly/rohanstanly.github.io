Key Features

Local AI Integration

Uses Ollama to run DeepSeek-R1 locally. Zero API costs, total data privacy, and works offline.

Interactive Visualization

Dynamic Line and Doughnut charts powered by Chart.js that update instantly when filters change.

Real-Time Data Management

Built on SQLite with a custom data entry system. Add sales and see the KPIs update in milliseconds.

Automated Reporting

Generates professional Markdown reports covering trends, risks, and strategies.

The AI Workflow

The core innovation of this dashboard is the "AI Intelligence Report." When a user selects a product or date range, the Python backend aggregates the raw SQL data and constructs a prompt. This is sent to a local Ollama instance running the DeepSeek-R1 model.

Figure 2: The AI generates a 4-part strategic report (Summary, Trends, Risks, Recommendations) entirely offline.

Granular Analytics

Beyond the high-level overview, the dashboard allows for deep dives into specific product performance. Each item in the inventory has its own mini-chart to visualize individual sales velocity.

Figure 3: Individual sparklines for every product allow managers to spot micro-trends instantly.

Data Management

The application is backed by a robust SQLite database. Users can view the raw transaction history to verify data integrity and audit specific sales.

Figure 4: The raw data view providing transparency into the last 100 transactions.

Technical Implementation

The connection between Flask and Ollama is handled via a simple REST API call. Here is the snippet that powers the AI generation: 

# Python Flask Backend connecting to Local Ollama
payload = {
    "model": "deepseek-r1:1.5b", 
    "messages": [{"role": "user", "content": prompt}],
    "max_tokens": 5000,
    "temperature": 0.6,
    "stream": False 
}

# Sending the request to localhost:11434
resp = requests.post("http://localhost:11434/v1/chat/completions", json=payload)
